## 9. 笔记历史记录

ENote 自动记录笔记的每次修改和删除操作历史，支持版本追溯和内容对比。

### 9.1 查看历史列表

**操作步骤：**

1. 选中一条笔记。
2. 点击工具栏右侧的"历史记录"按钮（时钟图标）。
3. 系统弹出全屏历史记录对话框，以表格形式展示历史记录列表。

历史记录表格包含以下列信息：

| 列名 | 说明 |
|------|------|
| ID | 历史记录的唯一标识 |
| 笔记本 | 笔记所属笔记本名称 |
| 标题 | 笔记标题 |
| 内容类型 | 富文本或 Markdown |
| 标签 | 关联的标签 |
| 操作类型 | 新增、修改或删除 |
| 操作来源 | 用户操作或 AI 工具（MCP） |
| 操作时间 | 操作发生的时间 |
| 操作 | 查看详情按钮 |

列表支持分页浏览，每页可选择显示 20、50、100 或 200 条记录。加载过程中显示骨架屏占位动画。

### 9.2 查看历史内容对比

在历史记录列表中，点击某条记录的"查看"按钮，打开内容对比查看窗口：

- **左侧面板（灰色标题）：** 显示修改前的旧内容。
- **右侧面板（主题色标题）：** 显示修改后的新内容。

两侧内容均以富文本格式渲染显示，方便直观对比修改差异。
