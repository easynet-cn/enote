## 24. 自动更新

ENote 支持应用内自动更新，通过 GitHub Releases 分发新版本。更新包经过数字签名验证，确保安全可靠。

### 24.1 启动时自动检查

应用启动后会自动在后台检查是否有新版本可用：

- 启动 3 秒后静默检查，不影响正常使用
- 如果没有新版本，不会弹出任何提示
- 如果发现新版本，会弹出更新对话框
- 检查结果会记录在操作日志中（模块：`updater`）

### 24.2 手动检查更新

除了自动检查，还可以手动触发更新检查：

- **菜单栏**：点击 帮助 → 检查更新
- **命令面板**：按 `Ctrl+P`（macOS: `Cmd+P`），搜索「检查更新」

手动检查时：
- 如果已是最新版本，会弹出「当前已是最新版本」提示
- 如果检查失败（如网络问题），会弹出错误提示

### 24.3 下载与安装

发现新版本时，更新对话框会显示：

- 当前版本号和新版本号
- 更新说明（Release Notes）
- **立即更新** 和 **稍后提醒** 两个选项

点击「立即更新」后：

1. 显示下载进度条
2. 下载完成后自动安装
3. 安装完成后提示重启
4. 应用自动重启并加载新版本

> **注意**：下载过程中请勿关闭应用。如果下载失败，可以稍后再次尝试。
