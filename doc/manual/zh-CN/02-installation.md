## 2. 软件安装与卸载

### 2.1 安装说明

**Windows 平台：**

1. 获取安装包文件（.msi 或 .exe 格式）。
2. 双击安装包，启动安装向导。
3. 阅读并同意软件许可协议。
4. 选择安装目录（默认为 C:\Program Files\enote）。
5. 点击"安装"按钮，等待安装完成。
6. 安装完成后，可在桌面或开始菜单找到 ENote 图标，双击启动。

**macOS 平台：**

1. 获取安装包文件（.dmg 格式）。
2. 双击 .dmg 文件，挂载磁盘映像。
3. 将 ENote 应用图标拖拽至"应用程序"文件夹。
4. 在"应用程序"中找到 ENote，双击启动。
5. 首次启动时，若系统提示安全确认，请前往"系统偏好设置 > 安全性与隐私"允许运行。

**Linux 平台：**

1. 获取安装包文件（.deb 或 .AppImage 格式）。
2. Debian/Ubuntu 系统：使用 `sudo dpkg -i enote_x.x.x_amd64.deb` 安装。
3. 其他发行版：赋予 AppImage 执行权限 `chmod +x enote_x.x.x.AppImage`，双击运行。

### 2.2 卸载说明

**Windows 平台：** 通过"控制面板 > 程序和功能"找到 ENote，点击"卸载"。

**macOS 平台：** 将"应用程序"中的 ENote 拖拽至废纸篓。

**Linux 平台：** Debian/Ubuntu 使用 `sudo apt remove enote` 卸载。
