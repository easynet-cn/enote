## 15. 命令面板

ENote 提供类似 VS Code 的命令面板功能，可通过键盘快速执行各种操作。

**打开方式：** 按 Ctrl+P（macOS 为 Cmd+P）快捷键打开命令面板。

**使用方法：**

1. 打开命令面板后，顶部出现搜索输入框。
2. 输入关键词，命令面板实时过滤并显示匹配的命令列表。
3. 使用方向键上下移动选择命令，按回车键执行。
4. 按 Esc 键关闭命令面板。

**可用命令：**

| 命令 | 说明 | 分类 |
|------|------|------|
| 新建笔记 | 创建一条新笔记 | 笔记 |
| 保存笔记 | 保存当前编辑的笔记 | 笔记 |
| 编辑笔记 | 进入编辑模式 | 笔记 |
| 删除笔记 | 删除当前笔记 | 笔记 |
| 展开/折叠侧边栏 | 切换侧边栏显示状态 | 视图 |
| 切换深色模式 | 切换浅色/深色主题 | 应用 |
| 打开设置 | 打开设置对话框 | 应用 |
| 打开回收站 | 打开回收站对话框 | 应用 |
| 数据备份 | 打开数据备份对话框 | 应用 |
| 模板管理 | 打开模板管理对话框 | 模板 |
| 从模板新建笔记 | 打开模板列表并使用模板创建笔记 | 模板 |
| 保存为模板 | 将当前笔记保存为模板 | 模板 |
| 锁定应用 | 立即锁定应用（需已开启锁屏功能） | 应用 |
| 切换布局模式 | 循环切换布局模式（自动→桌面→平板→手机） | 视图 |

命令按类别分组显示，便于快速查找。命令面板中显示的快捷键文本会根据用户自定义设置实时更新。
