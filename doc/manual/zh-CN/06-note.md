## 6. 笔记管理

### 6.1 笔记列表

笔记列表位于界面中间区域，展示当前筛选条件下的所有笔记。每条笔记卡片显示以下信息：

- **置顶标识：** 已置顶的笔记在卡片上显示置顶图标，并始终排列在列表顶部。
- **标题：** 加粗显示，单行截断。若未设置标题则显示"无标题"。
- **内容摘要：** 显示最多两行的纯文本预览，自动去除 HTML 和 Markdown 标记。
- **所属笔记本：** 左下角显示笔记归属的笔记本名称。
- **更新时间：** 右下角显示最后修改时间。

当前选中的笔记卡片会高亮显示。支持键盘方向键上下移动选择，按回车键或空格键确认选中。

**分页：** 列表底部提供分页控件，支持翻页浏览大量笔记。

**空状态：** 当没有笔记时显示引导提示"还没有笔记，点击创建新笔记"；搜索无结果时显示"没有找到匹配的笔记"。

**列表宽度调整：** 拖动笔记列表区域右侧边缘可调整列表宽度，支持 200 至 600 像素范围。

**列表面板折叠/展开：** 笔记列表右侧边缘中央有一个圆形折叠按钮，点击可完全折叠笔记列表面板，为编辑器腾出更多空间。再次点击可展开列表，展开后宽度恢复为默认的 320 像素。

### 6.2 新建笔记

**方式一：** 点击侧边栏顶部的"添加笔记"按钮（侧边栏展开时显示为完整按钮，折叠时显示为圆形加号图标）。

**方式二：** 使用键盘快捷键 Ctrl+N（macOS 为 Cmd+N）。

**方式三：** 使用命令面板（Ctrl+P），搜索并选择"新建笔记"命令。

新建笔记时：
1. 笔记自动归属于当前选中的笔记本。
2. 编辑器进入编辑模式，光标自动定位到标题输入框。
3. 用户需在工具栏左侧选择内容类型：
   - **富文本：** 使用所见即所得的富文本编辑器。
   - **Markdown：** 使用 Markdown 语法编辑器。

> **注意：** 笔记的内容类型在首次保存后不可更改。

### 6.3 编辑笔记

**操作步骤：**

1. 在笔记列表中点击选择要编辑的笔记，编辑器区域显示笔记内容（只读模式）。
2. 点击工具栏右侧的"编辑"按钮（铅笔图标），或使用快捷键 Ctrl+E（macOS 为 Cmd+E）进入编辑模式。
3. 编辑笔记标题和内容。
4. 编辑完成后点击"保存"按钮或使用快捷键 Ctrl+S（macOS 为 Cmd+S）保存。
5. 如需放弃更改，点击"取消"按钮或按 Esc 键。

> **注意：** 对于新建但尚未保存的笔记，点击"取消"后笔记将从列表中自动移除，不会保留空白笔记。对于已保存的笔记，取消编辑后内容恢复到上次保存的状态。

### 6.4 保存笔记

在编辑模式下，点击工具栏右侧的"保存"按钮（对勾图标）或使用快捷键 Ctrl+S（macOS 为 Cmd+S）保存当前笔记。保存成功后系统显示成功通知，笔记自动切换为只读模式。

系统会自动记录每次保存的历史版本，便于后续版本追溯和内容恢复。

### 6.5 删除笔记与回收站

**删除笔记（移入回收站）：**

1. 选中要删除的笔记。
2. 点击工具栏右侧的"删除"按钮（红色垃圾桶图标）。
3. 在确认对话框中点击"确认"执行删除。

删除后的笔记不会立即从数据库移除，而是被移入回收站（软删除）。

**回收站操作：**

点击侧边栏底部的"回收站"图标按钮打开回收站对话框，可以进行以下操作：

- **查看已删除笔记：** 回收站列表显示所有被删除的笔记，包含标题和删除时间。
- **恢复笔记：** 点击笔记右侧的"恢复"按钮，将笔记恢复到原来的笔记本中。
- **永久删除：** 点击"永久删除"按钮，彻底删除单条笔记，此操作不可恢复。
- **清空回收站：** 点击"清空回收站"按钮，永久删除回收站中的所有笔记。系统会弹出确认对话框，确认后执行清空操作，此操作不可恢复。

### 6.6 搜索笔记

笔记列表区域顶部提供搜索栏：

1. 在搜索框中输入关键词。
2. 按下回车键执行搜索。
3. 笔记列表将筛选显示标题或内容中包含关键词的笔记。
4. 搜索框右侧出现清除按钮（X 图标），点击可清除搜索条件并恢复完整列表。

搜索功能基于 SQLite FTS5 全文搜索引擎，使用 trigram 分词技术，支持中文子串匹配。搜索结果按相关性排序（BM25 算法），标题匹配权重高于内容匹配，确保最相关的笔记排在前面。

### 6.7 笔记设置

在编辑模式下，点击工具栏的"设置"按钮（齿轮图标）可打开笔记设置对话框，用于管理笔记的归属、标签和 AI 访问权限：

- **所属笔记本：** 通过可搜索的下拉选择器，将笔记移动到指定笔记本。
- **标签：** 以网格形式展示所有可用标签，以勾选框形式选中或取消关联的标签。一条笔记可关联多个标签。
- **AI 访问权限：** 设置 AI 工具通过 MCP 协议对该笔记的访问级别。可选值包括"继承"（默认，从标签和笔记本继承）、"读写"、"只读"、"禁止"。笔记级设置优先级最高，可以覆盖笔记本和标签的设置。详见 [20.4 权限控制](20-mcp.md#204-权限控制)。

修改完成后点击"保存"按钮生效。

### 6.8 笔记置顶

用户可以将重要笔记置顶，使其始终显示在笔记列表的最顶部。

**操作方式：**

- 在笔记列表中，右键点击笔记卡片或点击卡片上的置顶图标按钮，切换笔记的置顶状态。
- 已置顶的笔记在卡片上显示置顶标识。
- 再次点击可取消置顶。

置顶笔记在列表中排列在所有非置顶笔记之前，多条置顶笔记之间按更新时间排序。

### 6.9 笔记加密

ENote 支持对笔记内容进行 AES-256-GCM 加密，保护敏感信息的安全。

**加密笔记：**

1. 选中要加密的笔记，确保处于编辑模式。
2. 在工具栏中点击"加密"按钮。
3. 在弹出的加密对话框中输入密码，并再次确认密码。
4. 点击"加密"按钮，笔记内容将被加密保存。

**解密笔记：**

1. 打开一条已加密的笔记（编辑器会显示"此笔记内容已加密"的提示）。
2. 点击"解密"按钮。
3. 在弹出的对话框中输入加密时设置的密码。
4. 密码正确后，笔记内容恢复为明文显示。

> **注意：**
> - 加密使用 AES-256-GCM 算法，密钥通过 SHA-256 从用户密码派生，安全性高。
> - 请务必牢记加密密码，丢失密码将无法恢复笔记内容。
> - 加密后的笔记内容以密文形式存储在数据库中，无法通过搜索找到其原始内容。

### 6.10 关联笔记（双向链接）

ENote 支持在笔记之间建立双向链接关系，方便构建知识网络。

**查看关联笔记：**

在编辑器底部的"关联笔记"面板中，点击面板标题栏可展开/折叠面板。面板显示与当前笔记关联的所有笔记列表，包括笔记标题。点击任一关联笔记的标题可快速跳转查看该笔记。

**添加关联：**

1. 在关联笔记面板中，点击"添加链接"按钮。
2. 在弹出的搜索框中输入要关联的笔记标题关键词。
3. 从搜索结果列表中点击选择目标笔记，链接即刻建立。

**移除关联：**

将鼠标悬停在某条关联笔记上，点击右侧出现的"×"按钮即可移除该链接。

> **注意：** 双向链接是对称的——如果笔记 A 链接到笔记 B，那么在笔记 B 的关联面板中也会显示笔记 A。删除任一端的链接会同时移除双方的关联。

### 6.11 在新窗口中打开笔记

ENote 支持在独立窗口中打开笔记，方便同时查看或对比多条笔记。

**操作方式：** 通过命令面板或工具栏中的"在新窗口中打开"选项，将当前笔记在一个独立的应用窗口中打开。新窗口可独立调整大小和位置，具备完整的编辑功能。
