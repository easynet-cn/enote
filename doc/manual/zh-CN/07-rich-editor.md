## 7. 富文本编辑器

ENote 集成了功能丰富的所见即所得（WYSIWYG）富文本编辑器，工具栏位于编辑器顶部，当工具栏按钮过多时支持水平滚动，左右两侧有滚动指示箭头。

### 7.1 文本样式

工具栏提供以下文本格式化按钮：

| 按钮 | 功能 | 说明 |
|------|------|------|
| **B** | 粗体 | 将选中文本设为粗体 |
| *I* | 斜体 | 将选中文本设为斜体 |
| U | 下划线 | 为选中文本添加下划线 |
| ~~S~~ | 删除线 | 为选中文本添加删除线 |
| X² | 上标 | 将选中文本设为上标 |
| X₂ | 下标 | 将选中文本设为下标 |

点击按钮可切换对应格式的开启/关闭状态，选中文本后点击按钮应用格式，已应用的格式按钮会高亮显示。

### 7.2 标题与字体

**标题层级：** 通过下拉选择器设置段落的标题级别，支持：
- 正文（默认段落）
- 标题 1（H1）至 标题 6（H6），字号从大到小递减。

**字体族：** 下拉选择器提供多种字体选择，按类别分组：
- **无衬线体：** Arial、Helvetica、Verdana、Tahoma、Trebuchet MS、微软雅黑、苹方
- **衬线体：** Times New Roman、Georgia、Palatino、宋体、楷体、仿宋
- **等宽字体：** Courier New、Consolas、Monaco、Source Code Pro
- **艺术字体：** Comic Sans MS、Impact、Brush Script

**字体大小：** 下拉选择器支持 12px、14px、16px、18px、20px、24px、28px、32px、36px、48px、72px 等多种预设字号。

### 7.3 文本对齐

提供四种段落对齐方式按钮：

- **左对齐：** 文本靠左排列（默认）。
- **居中对齐：** 文本居中排列。
- **右对齐：** 文本靠右排列。
- **两端对齐：** 文本两端对齐排列。

### 7.4 列表功能

支持三种列表类型：

- **无序列表：** 以项目符号标记的列表。
- **有序列表：** 以数字编号标记的列表。
- **任务列表：** 带有复选框的任务清单列表，点击复选框可切换完成状态。已完成的任务项显示删除线和灰色文字效果。支持任务列表嵌套。

### 7.5 缩进控制

- **增加缩进：** 点击增加缩进按钮或按 Tab 键，段落向右缩进一级（每级 2em）。
- **减少缩进：** 点击减少缩进按钮或按 Shift+Tab 键，段落向左减少一级缩进。
- 最大缩进层级为 10 级，最小为 0 级。

### 7.6 引用与代码

- **引用块：** 点击引用按钮，将段落转为引用块样式，左侧显示主题色竖线，文本以灰色斜体显示。
- **代码块：** 点击代码块按钮，插入带语法高亮的代码块。支持多种编程语言的语法着色显示（基于 highlight.js 引擎）。
- **行内代码：** 点击行内代码按钮，将选中文本设为等宽字体的行内代码样式。

### 7.7 链接与图片

**插入链接：**
1. 选中需要添加链接的文本。
2. 点击工具栏的链接按钮。
3. 在弹出的对话框中输入链接地址（URL）。
4. 点击"确定"插入链接。链接文本以蓝色下划线样式显示。

**取消链接：** 选中已有链接的文本，点击"取消链接"按钮移除超链接。

**插入图片：**
1. 点击工具栏的图片按钮。
2. 在弹出的对话框中选择插入方式：
   - **输入 URL：** 在文本框中输入图片的网络地址。
   - **上传文件：** 点击上传按钮选择本地图片文件，支持常见图片格式。
3. 点击"确定"插入图片。

图片在编辑器中以圆角样式显示，支持懒加载技术，加载过程中显示动画占位符。

### 7.8 表格功能

- **插入表格：** 点击插入表格按钮，自动插入一个 3 行 3 列的表格。
- **添加列：** 在当前列后方插入新列。
- **添加行：** 在当前行下方插入新行。
- **删除表格：** 删除整个表格。
- 表格支持通过拖拽列边框调整列宽。选中的单元格会高亮显示。

### 7.9 颜色设置

**文字颜色：** 点击文字颜色按钮打开颜色选择器，提供 10 种预设颜色的宫格选择，同时支持通过颜色选择器或输入十六进制色值自定义颜色。

**背景颜色：** 点击背景颜色按钮，为选中文本设置背景高亮色，颜色选择方式同文字颜色。

**荧光笔高亮：** 点击高亮按钮，快速为选中文本添加黄色高亮背景。

### 7.10 查找与替换

点击工具栏的"查找替换"按钮，在编辑器上方打开查找替换面板：

- **查找输入框：** 输入要查找的文本，按回车键定位到下一个匹配项。
- **替换输入框：** 输入替换后的文本。
- **区分大小写：** 复选框控制是否区分大小写匹配。
- **结果计数：** 显示当前匹配位置和总匹配数（如"3 / 12"）。
- **上一个/下一个：** 箭头按钮在匹配结果间导航。
- **替换：** 替换当前匹配项。
- **全部替换：** 一次性替换所有匹配项。

所有匹配项在编辑器中以黄色高亮显示，当前聚焦的匹配项以橙色高亮区分。

### 7.11 目录导航

点击工具栏的"目录"按钮，在编辑器右侧打开浮动目录导航面板（宽度 240 像素）：

- 自动提取文档中的所有标题（H1-H6），按层级缩进展示。
- 点击目录中的任一条目，编辑器自动滚动定位到对应的标题位置。
- 当前可视区域内的标题在目录中高亮显示（左侧显示主题色竖线）。
- 点击面板右上角的关闭按钮（X）关闭目录面板。

### 7.12 撤销与重做

- **撤销：** 点击撤销按钮，回退到上一步编辑操作。
- **重做：** 点击重做按钮，恢复被撤销的操作。

### 7.13 清除格式

点击工具栏的"清除格式"按钮，移除选中文本的所有格式化样式，恢复为默认文本格式。

### 7.14 内容块拖拽排序

富文本编辑器支持通过拖拽方式调整内容块的顺序。将鼠标悬停在段落、标题、引用块、代码块等块级元素上时，元素左侧会出现拖拽手柄。按住拖拽手柄并拖动，可将该内容块移动到文档中的其他位置。拖动过程中，编辑器会显示当前拖拽状态的视觉指示，释放鼠标后内容块将插入到目标位置。

此功能适用于快速调整文档结构，无需通过剪切粘贴操作即可重新排列内容。

### 7.15 数学公式

富文本编辑器支持数学公式的渲染显示，基于 KaTeX 数学排版引擎。用户可以在笔记中插入数学公式，系统自动将公式语法渲染为标准的数学符号格式，支持行内公式和独立公式块，满足学术笔记和技术文档的数学表达需求。

### 7.16 图片懒加载

编辑器中的图片采用智能懒加载技术。当笔记包含较多图片时，仅加载当前可视区域内的图片，未进入可视区域的图片显示为带有动画效果的骨架占位符。当用户滚动页面使图片进入可视范围时，图片自动开始加载并以渐入动画显示。此机制有效减少初始加载时间，提升长文档的浏览流畅度。

### 7.17 智能粘贴

编辑器具备智能粘贴功能，自动清理从外部来源（如 Microsoft Office、Google Docs、网页等）复制粘贴的 HTML 内容：

- **清理 Office 标记：** 自动移除 MS Office 特有的 XML 命名空间、`mso-` 样式前缀等冗余标记。
- **清理网页格式：** 移除多余的 CSS 类名、`data-*` 属性，仅保留基本的文本格式（粗体、斜体、颜色等）。
- **标签转换：** 将 `<div>` 等块级元素自动转换为标准的 `<p>` 段落。
- **去除空元素：** 自动移除无内容的空段落和空 `<span>` 标签。
- **截图粘贴：** 支持直接粘贴剪贴板中的截图图片，图片自动保存为本地文件。

### 7.18 图片本地存储

编辑器中通过粘贴或拖拽插入的图片会自动保存为本地文件，而非以 Base64 编码内联存储在笔记内容中。图片文件保存在应用数据目录下的 `images/` 子目录中，每张图片使用 UUID 作为文件名，支持 PNG、JPG、GIF、WebP、SVG、BMP、ICO 等常见格式。

本地文件存储的优势：
- **减小数据库体积：** 笔记内容不再包含大量 Base64 编码的图片数据。
- **提升加载性能：** 图片通过 Tauri asset protocol 直接从本地文件系统加载，速度更快。
- **便于管理：** 图片文件独立存储，可直接在文件系统中查看和管理。
