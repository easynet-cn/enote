## 17. 键盘快捷键

ENote 提供以下键盘快捷键，提升操作效率：

| 快捷键 | 功能说明 | 可自定义 |
|--------|---------|---------|
| Ctrl/Cmd + S | 保存当前笔记 | 否（系统级） |
| Esc | 取消编辑 / 关闭命令面板 | 否（系统级） |
| Ctrl/Cmd + N | 新建笔记 | 是 |
| Ctrl/Cmd + E | 进入编辑模式 | 是 |
| Ctrl/Cmd + B | 展开/折叠侧边栏 | 是 |
| Ctrl/Cmd + P | 打开命令面板 | 是 |
| Ctrl/Cmd + L | 锁定应用（需已开启锁屏功能） | 是 |
| Ctrl/Cmd + Shift + M | 循环切换布局模式 | 是 |
| Tab | 增加缩进（编辑器内） | — |
| Shift + Tab | 减少缩进（编辑器内） | — |
| 方向键 ↑/↓ | 在笔记列表中上下移动焦点 / 在命令面板中选择命令 | — |
| Enter/空格 | 选中笔记列表中的聚焦项 / 执行命令面板中的命令 | — |

> **注意：** macOS 系统下 Ctrl 键对应 Cmd 键。标记为"可自定义"的快捷键可在设置中修改绑定，详见 [14.4 快捷键设置](14-settings.md#144-快捷键设置)。
